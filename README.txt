## MandelBulb Viewer

## Building and Running
This should only be built with Visual Studio 2012 in 64bit-mode.
Simply open the solution with vs2012, build, and run.

## Controls
The mouse is used to look around the world.

- Forward:    w
- Back:       s
- Left:       a
- Right:      d
- Adjust Speed:   Mouse wheel
- Enable fullscreen: f
